# Changelog

## 2026-06-13

- Added a playable UNO web app under `apps/uno/`.
- Added a dark blue game-table theme.
- Added David Edition card backs.
- Added action-card symbols for Skip, Reverse, Draw cards, Wild, Discard All, Color Roulette, and Flip.
- Added bot difficulty settings: Easy, Medium, and Hard.
- Added game modes for Classic, No Mercy, and Flip.
- Added No Mercy-style forced drawing, 0/7 hand swaps, Discard All, Skip Everyone, Color Roulette, larger draw cards, and 25-card mercy elimination.
- Added Flip-style light and dark sides, Flip cards, dark-side colors, Draw Five, Skip Everyone, and Wild Draw Color.
- Changed the win popup so clicking `You Win!` dismisses the popup.
