# davidclim.github.io

This folder is the local source for the GitHub Pages site at:

https://davidclim.github.io

## Local workflow

1. Create or edit web apps in this folder.
2. Put standalone apps under `apps/<app-name>/`.
3. Link each app from `index.html`.
4. Commit and push changes to GitHub.
5. GitHub Pages publishes the site from the repository.

## Expected GitHub repository

Create or connect this folder to a GitHub repository named:

```text
davidclim.github.io
```

For a user/organization Pages site, GitHub expects the repository name to exactly match the web address.
